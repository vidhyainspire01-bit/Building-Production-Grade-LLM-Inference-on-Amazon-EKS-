# GenAI Multi-Agent Platform (LangGraph)

This repository contains the unified multi-agent platform powering CUE, SDLC Automation, and future GenAI workflows.

## Features
- Separate isolated workflows (CUE, SDLC)
- LangGraph-based agent orchestration
- Workflow-level memory via checkpointers
- Tool registry for RAG + GitHub + Slack + TF
- Production-ready structure
